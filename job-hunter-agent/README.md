
#구직 AI 에이전트

## CrewAI 사용 
- job hunter agent는 유저의 이력서(resume)가 필요하기 때문에 knowledge > resume.txt에 관련정보를 작성함
- knowledge는 Agent가 쓸 지식을 미리 넣어둠
- https://docs.crewai.com/en/concepts/knowledge


## Firecrawl 사용
- 웹에서 차단 당하지 않고 검색할 때 무엇을 보여줄때 사용
- 웹을 스크랩 할 때 사용할 수 있음
- AI agent 를 만들 때 막히지 않고 활용 가능 
- 예제) https://www.firecrawl.dev/playground?endpoint=scrape
